# Thumbnail Planning

YouTube 썸네일의 컨셉을 기획하는 스킬. 클릭률(CTR)을 극대화하는 시각적 구성을 설계한다.

## 워크플로우

### 1. 입력 확인

다음을 수집한다:
- 완성 스크립트 (`_workspace/02_writer_script.md`)
- 제목 후보 (`_workspace/03_seo_optimization.md`에서 읽거나, SEO Optimizer로부터 수신)
- 채널 기존 썸네일 스타일 (있으면)

### 2. 핵심 메시지 추출

스크립트와 제목에서 썸네일이 전달해야 할 핵심 메시지를 추출한다:
- 영상의 가장 강렬한 포인트는 무엇인가
- 시청자가 느낄 감정은 무엇인가 (놀람, 호기심, 공포, 기대)
- 한 장의 이미지로 이 영상을 왜 봐야 하는지 전달할 수 있는가

### 3. 컨셉 설계 (최소 2개)

각 컨셉에 대해 다음을 상세히 기술한다:

**레이아웃:**
- 전체 구도 (3분할, 중앙 집중, 좌우 대비 등)
- 시선 유도 흐름 (사용자의 눈이 어디에서 어디로 이동하는가)

**비주얼 요소:**
- 배경 (색상/장면/그라데이션)
- 주요 오브젝트/인물 배치와 크기
- 표정/감정 (인물이 있는 경우)
- 아이콘/일러스트/화살표 등 보조 요소

**텍스트 오버레이:**
- 문구 (3~5단어 이내)
- 폰트 스타일 (굵기, 크기, 색상)
- 위치와 배경 처리 (그림자, 배경 박스 등)

**색상 팔레트:**
- 주색 + 보조색 + 강조색
- YouTube 피드에서 눈에 띄는지 (빨간 UI와 대비)

### 4. 산출물 저장

`_workspace/04_thumbnail_concepts.md`에 저장한다:

```markdown
# 썸네일 컨셉 기획서

## 영상 정보
- 제목: [선택된 제목]
- 핵심 메시지: [1문장]
- 타겟 감정: [놀람/호기심/...]

## 컨셉 A: [컨셉명] ⭐ 추천
- **레이아웃**: ...
- **배경**: ...
- **인물/오브젝트**: ...
- **텍스트**: "[문구]" — [폰트/색상/위치]
- **색상 팔레트**: 주색 #XXX, 보조 #XXX, 강조 #XXX
- **추천 이유**: ...

## 컨셉 B: [컨셉명]
...

## 제작 가이드
- 해상도: 1280x720 (최소), 권장 1920x1080
- 안전 영역: 우하단 타임스탬프 영역 피하기
- 파일 형식: JPG 또는 PNG, 2MB 이하
```

## 원칙

- 모바일 퍼스트. 작은 화면에서 인식 가능한 단순한 구성이 최우선
- 텍스트는 제목을 반복하지 않는다. 제목을 보완하는 감정적/추가적 정보를 담는다
- 감정 > 정보. 썸네일의 역할은 정보 전달이 아니라 클릭 충동 유발
- 채널 일관성을 유지하되, 매너리즘에 빠지지 않는 변화를 준다
