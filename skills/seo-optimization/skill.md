# SEO Optimization

YouTube 영상의 검색 노출과 클릭률을 극대화하는 메타데이터를 설계한다.

## 워크플로우

### 1. 입력 수집

다음을 확인한다:
- 완성 스크립트 (`_workspace/02_writer_script.md`)
- 주제 및 트렌드 데이터
- 채널 정보 (니치, 기존 영상 스타일)

스크립트의 "핵심 키워드" 섹션을 시드 키워드로 활용한다.

### 2. 키워드 분석

웹 검색을 통해 키워드를 확장하고 분석한다:

- **주요 키워드**: 검색량 높고 영상 주제를 정확히 반영
- **롱테일 키워드**: 경쟁 낮고 구체적인 검색 의도 반영
- **관련 키워드**: 주제와 연관된 확장 키워드
- **트렌드 키워드**: 현재 상승세인 검색어

### 3. 제목 최적화

제목 후보 3개를 작성한다. 각 제목은:

- 주요 키워드를 앞쪽에 배치
- 60자 이내 (잘리지 않게)
- 클릭 유발 요소 포함 (숫자, 의문, 감정, 대비)
- 낚시성이 아닌 실질적 가치 약속

**제목 공식 예시:**
- "[숫자] + [키워드] + [결과/혜택]" → "5가지 습관으로 영어 발음 완전 교정"
- "[키워드] + [반전/놀람]" → "ChatGPT 이렇게 쓰면 오히려 손해"
- "[대상] + [키워드] + [방법]" → "초보 개발자가 연봉 5천 올리는 현실적 방법"

### 4. 태그 구성

25~30개 태그를 우선순위순으로 구성한다:

1. **핵심 태그** (5개): 영상 주제를 정확히 반영
2. **롱테일 태그** (10개): 구체적 검색 쿼리
3. **카테고리 태그** (5개): 니치/장르 분류
4. **관련 태그** (5~10개): 연관 주제, 트렌드 키워드

### 5. 설명문 작성

- **첫 2줄** (접힌 상태): 핵심 키워드 + 영상의 핵심 가치 (사용자가 "더보기"를 누르게)
- **본문**: 영상 내용 요약 + 타임스탬프 + 관련 링크
- **하단**: 소셜 미디어 링크, 채널 소개

### 6. 산출물 저장

`_workspace/03_seo_optimization.md`에 저장한다:

```markdown

2. [제목] — 근거: ...
3. [제목] — 근거: ...

## 태그 (우선순위순)

## 해시태그

## 추가 권장사항
- 카테고리: [추천 카테고리]
- 게시 시간: [추천 시간대 + 근거]
- 자막: [권장 여부]
```

## 원칙

- 검색 노출(SEO)과 클릭률(CTR)을 동시에 최적화한다. 둘 중 하나만 잡으면 실패
- 낚시 제목은 장기적으로 채널을 해친다. 콘텐츠가 전달하는 실질적 가치를 제목에 담는다
- 태그는 너무 광범위(예: "유튜브")하거나 너무 좁지 않은 적정 범위를 찾는다
- 제목 후보를 Thumbnail Planner에게 공유하여 시각-텍스트 일관성을 확보한다
